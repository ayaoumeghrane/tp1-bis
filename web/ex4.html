Listes en HTML

Les etapes d’un projet site web sont :

  1. Analyse des besoins
  2. Conception du design
  3. Développement du site
  4. Tests et validation
  5. Mise en ligne

Outils

  • Éditeur de code
  • Navigateur web
  • Langages
    • HTML / CSS
    • JavaScript
  • Serveur web

Glossaire

HTML
Langage de balisage servant à structurer le contenu d’une page web.
CSS
Langage de style utilisé pour la mise en forme et la présentation.
JS (JavaScript)
Langage de programmation permettant d’ajouter de l’interactivité aux pages web.
